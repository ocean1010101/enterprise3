**Campus Companion Web Application**


Chosen Stack:

&#x20;- Next.js

&#x20;- TypeScript

&#x20;- Tailwind CSS

&#x20;- Lucide React

&#x20;- Local JSON

&#x20;- Netlify

&#x20;- PNPM




How to run locally:

Cloned GIT Repository - "git clone (repo link)"

&#x20;> cd enterprise3/campus-companion


I installed PNPM - "pnpm install"


I ran "pnpm dev"

&#x20;> copy and paste localhost link into your browser to see the result of your web app.



**Build command and Setup**

The application is deployed through Netlify. 
It uses continous deployment thats connected to my github repository. 
Everytime i do "git push origin main" this triggers an automatic build and redployment of the web application.

The web app uses a local "seedData.json" file for data management. 
There are no API keys or database credentials from supabase needed to run the application.
